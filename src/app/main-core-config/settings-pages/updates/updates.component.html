Binary Updates Controls where Particl Core binary updates are obtained from. This is only applicable if a new Particl Core node is created/started from Particl Desktop.

Automatically Update Particl Core

Enabling this allows for Particl Core to perform a check for new available versions prior to starting a new Particl Core node.

Disabling this may prevent the correct Particl Core binary from being downloaded when attempting to start a new Particl Core node for the first time. It is recommended to leave this enabled to ensure that Particl Core is always at the latest recommended version.

Allow for Particl Core to automatically update

Download Timeout (#seconds)

When downloading a new Particl Core binary, specify how long to wait, in seconds, for the download to complete before abandoning the download.

This value can be increased if using a slow internet/network connection.
