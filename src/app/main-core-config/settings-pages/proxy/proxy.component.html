Proxy / Tor Configuration Particl Core can be configured to connect via a proxy with other Particl nodes. Typically this might be the Tor network, an open-source annonymous communications network that aims to protect the privacy, freedom and confidentiality of its users through IP address annonymity.

Use a (Tor) proxy

Instruct Particl Core to connect via a proxy.

Note that this selection only applies when starting a new Particl Core node. If Particl Core is already running, it will need to be stopped or disconnected from, and a new node started afterwards, in order to to take effect.

Use a (Tor) Proxy

Proxy Address

The address of the proxy to connect through.

The default Tor proxy address is provided as a default option, but if it has been customized, or an alternative proxy is being used, the address below should be edited accordingly.
